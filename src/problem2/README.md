## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

* NodeJS (>18.0.0)

* pnpm (9.x.x)

### Development

Install dependencies

```
pnpm install
```

And finally just start the application
```
pnpm dev
```
